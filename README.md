# PostgreSQL开发指南

配套视频：https://www.bilibili.com/video/BV1ZW4y127cz

《PostgreSQL开发指南》PDF版本以及示例表和初始化数据。

Scripts目录中的脚本执行顺序：

1. 运行01_create_table.sql创建示例表;
2. 运行02_insert_data.sql生成数据；
3. 运行03_create_index.sql创建索引；
4. 运行99_delete_table.sql删除示例表。
